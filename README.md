      Key       │ Description                                  
───────────────┼──────────────────────────────────────────────
 Ctrl-h         │ Move to left window                          
 Ctrl-j         │ Move to window below                         
 Ctrl-k         │ Move to window above                         
 Ctrl-l         │ Move to right window                         
 \e             │ Toggle file tree                             
 a              │ Create file or directory                     
 r              │ Rename                                       
 x              │ Cut                                          
 c              │ Copy                                         
 y              │ Copy name/path                               
 p              │ Paste                                        
 d              │ Delete                                       
 D              │ Trash (if enabled)                           
 e              │ Rename base only                              
 s              │ Open file system shell (if enabled)          
 f              │ Filter by string                             
 F              │ Clear filter                                 
 I              │ Toggle hidden files                           
 H              │ Toggle dotfiles                               
 R              │ Refresh tree                                 
 S              │ Search (live grep)
